# 🚗 ระบบบันทึกค่าใช้จ่ายยานพาหนะ

## การติดตั้ง

### ข้อกำหนดเบื้องต้น
- **Node.js** เวอร์ชัน 16 ขึ้นไป ([ดาวน์โหลดที่นี่](https://nodejs.org))

### ขั้นตอนการติดตั้ง

1. เปิด **Command Prompt** หรือ **Terminal** ในโฟลเดอร์นี้
2. รันคำสั่ง:
   ```
   npm install
   ```
3. เริ่มระบบ:
   ```
   npm start
   ```
4. เปิดเบราว์เซอร์ไปที่ `http://localhost:3000`

### การเข้าถึงจากมือถือ
- เชื่อมต่อมือถือกับ WiFi เดียวกับเครื่อง PC
- หา IP ของ PC (รัน `ipconfig` บน Windows)
- เปิดเบราว์เซอร์มือถือ: `http://<IP ของ PC>:3000`
  เช่น: `http://192.168.1.100:3000`

## การใช้งาน

### 1. ตั้งค่าเบื้องต้น (เมนู ⚙️ ตั้งค่า)
- เพิ่ม **หน่วยงาน** ก่อน (เช่น ฝ่ายขาย, ฝ่ายปฏิบัติการ)
- เพิ่ม **ชนิดรถ** ตามต้องการ

### 2. เพิ่มรถ (เมนู 🚙 จัดการรถ)
- กดปุ่ม "เพิ่มรถ"
- กรอกข้อมูลทะเบียน, หน่วยงาน, ชนิดรถ

### 3. บันทึกค่าใช้จ่าย (เมนู ➕ บันทึกค่าใช้จ่าย)
- เลือกหน่วยงาน → เลือกรถ → เลือกประเภทค่าใช้จ่าย
- กรอกจำนวนเงิน และข้อมูลเพิ่มเติม

### 4. ดู Dashboard (เมนู 📊 Dashboard)
- ปรับช่วงเวลา, หน่วยงาน, ชนิดรถ ตามต้องการ
- ดูกราฟและสรุปค่าใช้จ่าย

### 5. ออกรายงาน (เมนู 📈 รายงาน)
- เลือกช่วงเวลาและเงื่อนไข
- เลือกการจัดกลุ่ม (ตามประเภท/หน่วยงาน/ชนิดรถ/รถ/เดือน)

## ประเภทค่าใช้จ่ายที่รองรับ
- ⛽ ค่าน้ำมัน (บันทึกปริมาณและราคาต่อลิตรได้)
- 🔧 ค่าเซอร์วิสตามระยะ
- 🔨 ค่าซ่อม
- 🛞 ค่าเปลี่ยนยาง
- 🛡️ ประกันภัย
- 🛣️ ค่าทางด่วน/ค่าผ่านทาง
- 🅿️ ค่าจอดรถ
- 📋 พ.ร.บ. / ภาษีรถ
- 📝 ค่าใช้จ่ายอื่นๆ

---

## ไฟล์ในระบบ
- `server.js` — Backend server
- `public/index.html` — Frontend (UI)
- `vehicle_data.db` — ฐานข้อมูล SQLite (สร้างอัตโนมัติเมื่อรันครั้งแรก)
- `package.json` — ข้อมูล dependencies
